```
# 在后台启动服务
docker-compose up -d
```



1. 若要停止并移除服务，运行：

```
# 停止并移除容器
docker-compose down
```



### 常用管理命令

- **查看容器日志**：

  ```
docker logs -f minio-server
  ```
  
  

- **停止 MinIO 容器**：

  ```
docker stop minio-server
  ```
  
  

- **启动已停止的 MinIO 容器**：

  ```
docker start minio-server
  ```
  
  

- **删除 MinIO 容器**（请先停止它）：

  ```
docker rm minio-server
  ```
  
  

  **注意**：由于我们做了数据持久化，即使删除了容器，C:\minio-data 里的数据也不会丢失。下次重新创建容器并指向该目录，数据依然存在。
